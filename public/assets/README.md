# Assets Directory

## Логотип платформы

Поместите файл логотипа в эту папку с именем `logo.png`.

### Требования к логотипу:
- **Формат:** PNG (рекомендуется)
- **Имя файла:** `logo.png`
- **Размер:** Оптимально 256x256px или больше для retina дисплеев
- **Фон:** Прозрачный или подходящий под дизайн платформы

### Использование в коде:
```jsx
// В компонентах
<img src="/assets/logo.png" alt="Alpha Edge Logo" />

// В HTML
<img src="/assets/logo.png" alt="Alpha Edge Logo">
```

### Замена логотипа:
1. Скачайте ваш логотип
2. Переименуйте в `logo.png`
3. Поместите в папку `public/assets/`
4. Перезапустите dev сервер (`npm run dev`)

Файлы из папки `public/` доступны напрямую по URL без дополнительной обработки.
